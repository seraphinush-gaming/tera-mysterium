2016년 03월 10일 업데이트 안내

Thursday, March 10, 2016

1. 던전
n  환영의 탑
Ø  환영의 탑이 새로운 시즌 위상에 맞춰 재오픈 됩니다.
Ø  환영의 탑은 Lv.65 5인 파티플레이 던전입니다.
Ø  환영의 탑은 [웨스토니아 가드 > 얼어붙은 해안]에 위치한 NPC ‘피슝’ 또는 파티매칭을 통해 입장할 수 있습니다.
- 상세 위치는 ‘세계지도(단축키:M)’를 오픈하여 위 경로 대로 지도를 확대해 보시면 ‘환영의 탑 입구’ 위치를 확인해 보실 수 있습니다
Ø  환영의 탑은 1일 1회, VIP 2회 이용할 수 있습니다.
- 16층 이상의 입장 제한 횟수가 사라졌으며, 입장권 소지 시 제한 없이 입장 가능합니다.
Ø  환영의 탑에서는 10단계 강화석, 완전한 태고의 재봉인 주문서 10단계, 아티팩트 조각, 에메랄드, 오르피우스의 메달 등을 획득할 수 있습니다.
n  시험의 던전 (솔로 던전)
Ø  시험의 던전이 새로운 시즌 위상에 맞춰 재오픈 됩니다.
Ø  시험의 던전은 Lv.65 1인플레이 던전입니다.
Ø  시험의 던전은 이르카를 포함한 모든 대도시에 위치한 NPC ‘앵크’ 또는  [메뉴 UI > 던전 매칭 > 시험의 던전]을 통해 이용할 수 있습니다.
Ø  시험의 던전에서는 에메랄드, 다이아몬드, 보존의 가루 8단계, 완전한 태고의 재봉인 주문서 8단계 등을 획득할 수 있습니다.
Ø  또한, 시험의 던전 클리어 시 메달을 획득할 수 있으며, 메달을 통해 듀리온 액세서리, 베르노 장비, 아티팩트 원석, 철기장:복수 제작 재료를 구입할 수 있습니다.
n  천공의 경기장
Ø  천공의 경기장 입장 NPC ‘가데롱’을 모든 대도시에 추가 배치하였습니다.

n  정령사
Ø  순간 이동
- 쿨타임이 1초 감소합니다.
- 무적 시간이 소폭 증가합니다.
Ø  중형 몬스터에 대한 데미지가 소폭 감소합니다.

Ø  베르노의 실험실 던전(상/하급) 3네임드의 바닥 장판을 방어할 수 있는 문제를 수정하였습니다.

ü  스킬
Ø  창기사의 전선 유지 스킬 효과가 파티원이 아닌 플레이어에게도 적용되는 문제를 수정하였습니다.
Ø  검기술 스킬의 토글을 On 상태로 던전에 출입 시 '검기' 이상상태가 해제되며, '강력한 검기' 이상상태만 유지되는 문제를 수정하였습니다.
ü  UI
Ø  PC방 일일 누적 플레이 보상의 스트링이 잘려서 출력되는 문제가 수정되었습니다.
Ø  PC방 총 누적 플레이 40시간 보상의 수령 버튼이 '확인'으로 출력되는 문제를 수정하였습니다.
Ø  우편 UI를 ESC 키로 종료할 경우 UI 모드 전환 시 재출력 되는 문제를 수정하였습니다.
Ø  평판 포인트 획득 아이템 사용 시 화면 상단에 출력되는 메시지가 어색하게 출력되는 문제를 수정하였습니다.
Ø  퀘스트 알림UI에 퀘스트가 20개 이상 등록될 경우 퀘스트 번호가 비정상적으로 출력되는 문제를 수정하였습니다.
Ø  T-cat 상점에서 장바구니 선택 후 UI 재오픈 시 상점 카테고리 버튼이 출력되지 않는 문제를 수정하였습니다.
Ø  X를 조합키로 사용 시 특성 UI를 단축키로 오픈할 수 없는 문제를 수정하였습니다.
ü  문장
