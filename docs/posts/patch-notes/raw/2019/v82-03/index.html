  • 파르퀴나의 둥지 2네임드 전투공간에서 중앙/외곽 위치에 따라 일부 스킬 위력이 상이한 문제

  • 약탈자의 사막에서 보물지도 사용 시 론 NPC가 오브젝트에 걸려 이동이 중단되는 문제
