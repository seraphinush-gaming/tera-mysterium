1. 던전

  • 광기의 투기장
    • 불멸의 투사 회전 베기 스킬의 데미지가 감소됩니다.
    • 끌어당기기 패턴 사용 시, 불멸의 투사와 캐릭터가 겹치지 않도록 조정됩니다.

  • 엘린 월광무사로 종족 변경이 가능해집니다.

  • 광기의 투기장 불멸의 투사 어그로 대상이 어색한 문제
